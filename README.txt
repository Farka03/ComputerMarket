# Приложение "Компьютерный магазин"
Это небольшое приложение, созданное на Spring Boot, в основе которого находится магазин, продающий компьютеры и комплектующие к ним.

## Функциональность
Приложение предоставляет следующие основные возможности:
- Взаимодействие с in memory базой данных H2.
- Добавление товара.
- Редактирование товара.
- Просмотр всех существующих товаров.
- Просмотр товара по идентификатору.

## Зависимости
В этом проекте используются следующие зависимости:
- [spring-boot-starters]: предварительно упакованные наборы зависимостей.
- [H2]: in memory база данных.

## Установка и использование
Github:
1) Клонируйте репозиторий с Github.
2) Соберите приложение с помощью maven.
3) Настройке in memory базу данных H2 в application.properties.
4) Запустите приложение при помощи Maven.
5) Теперь можно проверить, что все данные сохраняются в бд - localhost:(порт)/h2 + логин и пароль, которые прописаны в .properties -> SELECT * FROM PRODUCT (к примеру). Для написания запросов рекомендую Postman.
